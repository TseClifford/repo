Installing the TseClifford Repo

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://tseclifford.github.io/repo
  • Name it "tseclifford" without the quotes
  • Head to " My Addons"
  • Select install an addon from zipfile.
  • When it asks for the location, select "tseclifford", and install repository.tseclifford-1.1.1.zip
  • Go back to My Addons and select "Install from repository"
  • Select the "TseClifford Repo" menu item
  • Open "Video add-ons" and install desired add-ons
